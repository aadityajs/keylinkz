This APP is an example of how to create an APP for phpFox and host it on your own server.

To install this APP...

1) Rename the file "config.php.sample" to "config.php".
2) Edit the settings in the file "config.php".
3) Open the file "mysql.sql" and run that query in phpMyAdmin.
4) Point your APP to the "index.php" file.
5) Look over the "index.php" to learn more on how to work with APPs.
